# AccountMigration.ps1 by Taehoon Song

This will move multiple multiple IDs into specified OU and update the user objects.

How to use:

1. Enter netID and TLA of the user in import.csv for its respective columns (enter TLA in all CAPS!) and save.
2. Run script. 
3. If you run into any errors, you can ignore for the most part, but it's a good idea to check the user object to make sure everything worked as intended.
